This is my first post!

As you have read from the title this is my first post, I have decided that for this special occasion I will show you all the things I can do with only HTML and a little of CSS. However I need to precise some fact about this website, the css is not perfect, the website in itself is not perfect, but I'm happy that I have reached to made this. The philosophy behind this website was inspired by Mother Fucking Website.

This website can do beautiful things without JavaScript or anything else. I'm not saying that JavaScript is bad, actually I think is good, for some case is really good. Unfortunately it is used many times in a wrong way for making things that can work perfectly without it, the most basic things of a website should be usable even without JavaScript.

Things I want to add in future

I must also say that this site will never be completed. I think I'll always be making changes in the future; I'm even considering adding JavaScript for a few small things. The style and some other things could also be improved, but let's remember that perfection doesn't exist.

Here is a short list of additions and changes I want to make:

  • Add feed rss support.
  • Improve and fix the CSS hambuger menu.
  • Add support to other language.
  • Improve accessibility.

There are other change that I have in mind and I want to apply to the site, maybe in the future. You can also view this project on GitHub, and if you want you can use it to create your own website. That's all for now.
